Bienvenue sur mon portfolio personnel, créé avec Vue.js, Tailwind CSS et Vite en 2023. Ce portfolio me présente en disant qui je suis, les technologies et outils que j'utilise au quotidien, mes différents projets que j'ai réalisé ainsi que mon parcours professionnel et scolaire.

- [Vue.js](https://vuejs.org/) - Framework JavaScript progressif
- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS utilitaire

## Comment exécuter localement

1. Clônez ce dépôt sur votre machine :

   ```bash
   git clone https://github.com/StarWeizz/Portfolio-Antonin-Russo-2023.git
   ```

2. Installez les dépendances nécessaires : 

    ```
    cd Portfolio-Antonin-Russo-2023
    npm install
    ```

3. Démarrez l'application en mode de développement :

    ```
    npm run dev
    ```

4. Ouvrez votre navigateur et accédez à l'URL suivante :

    ```
    http://localhost:5173
    ```

## Construire le projet 

1. Exécutez la commande pour créer les fichiers de production :

    ```
    npm run build
    ```

Vous y êtes ! Rendez-vous dans le dossier dist/ !
